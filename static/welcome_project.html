hdGSuite HyperBrowser: A web platform for high dimensional genomic data analysis

description

High dimensional genomic data

High dimensional genomic data can be presented as a matrix, with each column representing a sample (for example, a patient, a cell type, genotype, an experimental condition and so on), and each row representing a genomic tracks, which is represented by collection of features (for example, genes, genomic locus and so on).

In the project we represented high dimensional genomic data in a GSuite format. To read more about it look at GSuite specification. You can also look at the below example, where every single genomic track is represented by the set of characteristic defined by selected columns (genotype and mutation type).
This way of representing data allow us for effective analysis not only collections of single genomic data, but also analysis within groups and among groups.
Depend on the biological question, you can divide the same data into different groups. Let assume that you are interested in the statistic: average number of genomic elements per defined group such as genotype (left figure) or genotype and mutation type (right figure).
For genotype groups average number of genomic elements is counted for eta and iota group. For eta, iota we counted average number of genomic elements for tracks 1-3, 4-7 separately. For genotype and mutation type groups average number of genomic elements is counted for eta-CA, eta-CG, iota-CA, iota-CG and iota-CT group. For eta-CA, eta-CG, iota-CA, iota-CG, iota-CT we counted average number of genomic elements for tracks 1-2, 3, 4-5, 6, 7 separately.

Note:

This is a specialized version of The Genomic Hyperbrowser, focusing on new functionality for analyzing collections of genomic track. More information on the general HyperBrowser functionality can be found on the main project site.

Contact information

If you have any questions, requests or comments, please use our discussion forum (available from the Help menu) or send us an email to hyperbrowser-requests@usit.uio.no‎. We are happy to respond and welcome collaborative projects that help further develop our system. If you encounter any bugs, please let us know via an email to hyperbrowser-requests@usit.uio.no.

If you register as a user, you will be able to subscribe to the hyperbrowser-requests@usit.uio.no mailing list, which serves for distribution of system-related announcements and administrative messages. A backlog of announcements is available from the Help menu. The Help menu also contains citation information.

This project is being developed by the Norwegian bioinformatics community as an open-source project under the GPL license v3, supported by various national and local bodies.

For further information on Norwegian bioinformatics, please visit www.bioinfo.no.
