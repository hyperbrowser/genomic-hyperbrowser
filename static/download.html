Download Source Code

The source code to the Genomic HyperBrowser is available through an online SVN repository. To obtain the latest version, you need to make sure that SVN is installed on your computer. Then, please type the following: 

svn co svn://invitro.titan.uio.no/projects/bioinfoprojects/hyperbrowser_dist/trunk hyperbrowser_dist

The Genomic HyperBrowser needs an installation of Galaxy to install on top of. If you do not have an installation, please download one from the Galaxy web site in a different directory.

Follow the installation instructions included with the Genomic HyperBrowser distribution. Please note that the current version ($HB_VERSION) is synchronized against Galaxy version $GALAXY_VERSION, and needs this exact version in order to install.

Notice: Currently, the HyperBrowser is synced against a rather old version of Galaxy, with some security patches added. In order to install the correct version of Galaxy, please do the following: 

  hg clone https://bitbucket.org/galaxy/galaxy-dist/ -u 26920e20157f galaxy-dist-hb
  cd galaxy-dist-hb
  hg transplant -b default 115068e
  wget https://bitbucket.org/galaxy/galaxy-dist/changeset/b5bda7a5c34535ada63722941f2a2d62524b1faa/raw/ -O /tmp/galaxy_patch
  hg import /tmp/galaxy_patch
  hg summary
The Galaxy version should now be fa0b5c68d097 as required by HyperBrowser. In the next release of HyperBrowser, the Galaxy integration will be updated to a recent version.
